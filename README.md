## 개발환경 셋팅
에디터는 VisualStudio Code 를 이용합니다.
문법관리는 eslintrc 를 이용해 관리합니다. (후에 추가 예정)...

#### 필수 의존성 모듈 설치
1. node 를 설치합니다. (v10.16.0)
2. yarn 를 설치합니다. (`npm install yarn -g`)
3. react 를 설치합니다. (`yarn add react -g`)

#### 리포지토리 클론
1. git clone [리포지토리 url]
2. `cd [리포지토리 이름] && ` (`yarn`)

## 실행 방법
#### 프론트엔드
`yarn start`
